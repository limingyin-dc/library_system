# 图书馆系统启动指南

## 环境准备

确保已激活虚拟环境：

```bash
venv\Scripts\activate
```

## 执行步骤

### 1. 生成数据库迁移文件

```bash
python manage.py makemigrations accounts books borrow catalog purchase readers
```

### 2. 应用迁移到数据库

```bash
python manage.py migrate
```

### 3. 创建管理员账号

```bash
python manage.py createsuperuser
```

按提示输入用户名、邮箱（可留空）、密码。

### 4. 启动开发服务器

```bash
python manage.py runserver
```

## 访问地址

| 页面 | 地址 |
|------|------|
| 系统首页 | http://127.0.0.1:8000 |
| 后台管理 | http://127.0.0.1:8000/admin |

## 常见警告说明

- `staticfiles.W004`：static 目录不存在，不影响运行，创建 `static/` 文件夹可消除该警告。
